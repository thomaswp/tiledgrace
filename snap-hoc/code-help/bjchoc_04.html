Step 5: Make Alonzo move more often when he's not clicked

The idea for this step is to have the time Alonzo waits before moving get smaller each time it's clicked (to make the game more challenging). The delay will start at 2 and drop by 0.1 second with every click.

To do this, you can create a variable called delay with the command var delay := 2. This names your variable delay, and sets it to start at 2 seconds. Make sure to place it at the beginning of your code, so they variable is in scope throughout the rest of your code. You can use delay anywhere you could use a number or a command that returns a value. You can also set its value with delay := ... or use it in a calculation, like x - 0.1.
